# על האתר

האתר הזה הוא תבנית התחלה לסדנת מורים קצרה: מאגר GitHub ציבורי, דפי Markdown בעברית, ופרסום אוטומטי דרך GitHub Pages.

המטרה היא לאפשר תחזוקה פשוטה:

- כל עמוד הוא קובץ Markdown רגיל.
- הניווט נקבע באמצעות שדות front matter.
- העיצוב מגיע מגרסת Just the Docs נעולה.
- התאמות עברית נמצאות בקובץ Sass מקומי קטן.

{: .task }
לפני שמעלים חומרי לימוד אמיתיים, ודאו שאין בקבצים מידע פרטי של תלמידים, הורים, צוות או בית הספר.

## מבנה בסיסי

```text
.
├── index.md
├── about.md
├── lessons/
│   ├── markdown-basics.md
│   └── organizing-lessons.md
└── _sass/
    └── custom/
        └── custom.scss
```

אפשר להתחיל מהשיעורים לדוגמה, לשנות אותם, או להוסיף שיעורים חדשים לצידם.
