# ערכת שיעורים בעברית

ברוכים הבאים לאתר לימודי קטן שאפשר להרחיב בקלות באמצעות קובצי Markdown. האתר מיועד למורה שרוצה לפרסם חומרי שיעור ברורים, קצרים ונגישים, בלי לנהל קוד מורכב או סביבת פיתוח מקומית.

{: .note }
האתר נבנה כמאגר תוכן נקי: מוסיפים עמודים, דוחפים ל-GitHub, ו-GitHub Pages מפרסם את הגרסה המעודכנת.

## דוגמת קוד קצרה

קוד, שמות קבצים ופקודות נשארים בכיוון שמאל-לימין כדי שיהיה קל להעתיק ולקרוא אותם:

```bash
git add index.md
git commit -m "Add lesson content"
git push
```

## איך ממשיכים?

כדי להוסיף שיעור חדש, יוצרים קובץ Markdown בתיקיית `lessons/`, מוסיפים front matter בראש הקובץ, וכותבים את תוכן השיעור בעברית.
