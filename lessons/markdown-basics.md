# יסודות Markdown

Markdown מאפשר לכתוב תוכן לימודי בפשטות: כותרות, רשימות, טבלאות וקישורים, בלי לערוך HTML ידנית.

{: .note }
מומלץ שכל שיעור יתחיל במטרה קצרה, ואחריה פעילות אחת או שתיים שאפשר לבצע בכיתה.

## מטרות השיעור

- להכיר כותרות ורשימות.
- להבין איך מוסיפים קישור פנימי באתר.
- להתנסות בקטע קוד קצר.

## דוגמה לכתיבה

```markdown
## כותרת משנה

- רעיון ראשון
- רעיון שני

[חזרה לדף הבית]({{ '/' | relative_url }})
```
